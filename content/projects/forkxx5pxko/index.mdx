# FORK
**Project ID:** `xX5pxKo`  
**Hub:** Cardano Hub NBO — Nairobi, Kenya

## Problem Statement
Prepaid health funds in Kenya could be managed more effectively to improve access, accountability, and trust across patients, clinics, and sponsors.

## Team Members
- Owen Nyangweso Nyabicha - Frontend & Product lead (Responsible for owning the frontend architecture and user experience, translating product requirements into clear UI flows, and ensuring consistent, reliable interaction with backend APIs across all user roles.  Accountable for frontend code quality, state management, API integration correctness, and timely delivery of user-facing features.  Ensures the frontend reflects backend state accurately, handles asynchronous operations gracefully, and maintains a predictable, usable experience for patients, clinics, and admins.)
- Kelvin Wepo - Backend Engineer- Integrations & platform support (Responsible for user authentication, role management, clinic onboarding, settlement tracking, and system-level services supporting the core backend.  Accountable for API documentation, environment configuration, and maintaining reliable integration points for frontend and external services.  Ensures the system remains observable, configurable, and operable as the product evolves.)
- Francis Masila - Fullstach developer- core systems & ledger (Responsible for designing and implementing the core backend domain logic, including wallets, health credits, visit consumption, and the internal ledger.  Accountable for data integrity, balance correctness, rule enforcement, and ensuring all state transitions are valid, auditable, and consistent.  Ensures the backend acts as the single source of truth and is structured to support future blockchain anchoring without architectural rework.)
