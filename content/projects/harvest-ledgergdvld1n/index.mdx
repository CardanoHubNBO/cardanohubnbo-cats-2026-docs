# Harvest Ledger
**Project ID:** `gDVld1N`  
**Hub:** Cardano Hub NBO — Nairobi, Kenya

## Problem Statement
Consumers overpay for fraudulent "organic" food with no proof. HarvestLedger provides QR-verified authenticity & direct farmer connection via Cardano.
